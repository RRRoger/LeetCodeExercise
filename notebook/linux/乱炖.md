## JavaScript 时间处理的函数

```javascript
/**
* 字符串转时间(`yyyy-MM-dd HH:mm:ss` | `yyyy-MM-dd`)
* 暂时只满足以上两种格式
*/
String.prototype.rogerStringToDate=function(){
    t = this.split(/[-\ :]/);
    t[1]--;
    return new Date(...t);
};
```

## pgsql 字符串操作 

```sql
--把字段outer_code以.0结尾的 替换成字符串 达到去除.0的效果
update product_product set outer_code = replace(outer_code,'.0','') where outer_code like '%.0'
```
